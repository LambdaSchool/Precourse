for: used with for loop.
Followed by conditions and will continue to loop through until conditions are met
example:
for(var i = 1; i < 10; i++) {
do this;
}

&&: and. used in expressions
example:
if (exp is true && exp is true) {
do this;
}

||: or. used in expressions
example:
if (exp is true || exp is true) {
do this;
}

!: not. used in expressions
example:
if (exp !true) {
do this;
}
