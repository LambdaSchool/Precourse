for
-- the keyword for is used to indicate a for loop. A for loop repeats the action indicated within the curly
braces until the portion of code inside the parentheses is false. 


&&
--&& indicates AND -- it will return output only if both statements on either side of && will return true

||
--|| indicates OR -- it will return output when either one of the statements is true

!
--indicates NOT -- will return the opposite of what is written
