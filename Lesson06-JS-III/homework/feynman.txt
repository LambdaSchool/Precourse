	* Arrays

	an array is an ordered container that stores groups of values. Typically those values will all be related or the same type but they do not have to be

		let example = ['forty five', 46]
	to access a value from the array you use the variable name plus the index number
	remember that index numbers begin with 0
		example[0] returns 'forty five'
		example[1] returns 46
